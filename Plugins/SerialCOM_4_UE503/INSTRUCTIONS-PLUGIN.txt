
BLUEPRINT FOR UNREAL ENGINE (4.25/4.26/4.27/5.0.3)
-------------------------------------------------------------

#1 Select the Zipfile for your correct Engine:
	- SERIALCOMUE425/SERIALCOM_UE425.zip 	(Unreal Engine 4.25)
	- SERIALCOMUE426/SERIALCOM_UE426.zip 	(Unreal Engine 4.26)
	- SERIALCOMUE427/SERIALCOM_UE427.zip 	(Unreal Engine 4.27)
	- SERIALCOMUE5EA/SERIALCOM_UE5EA_r1.zip (Unreal Engine 5EA)
--------------------------------------------------------------------
	- SERIALCOM_4_UE503/SERIALCOM_4UE503.zip (Unreal Engine 5.0.3)




#2 Unzip the correspondent Zip file into your project "Plugins" Folder. If doesn't exist, create one.
---------->Your_Project_Folder-->
----------------->Binaries
----------------->Config
----------------->Content
----------------->DerivedDataCache
----------------->Intermidiate
----------------->Plugins_
                          |------>SERIALCOM_EUXXX
----------------->Saved
Your_Project.uproject

#3 Activate  "Communication Serial Port (Serial COM)" plugin from your plugin list.

#4 Re-Start your Project. You are ready to use the plugin!!!
